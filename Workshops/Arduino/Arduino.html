Arduino

Arduino Arduino is an open-source platform of hardware (i.e., micro-controller, sensors) and software used to build prototypes that can run without the assitance of a computer and interact with other software on a computer.


Link to Arduino files

Blink (3B)


The circuit is built using two LEDs, resistors, four jumpers, a board and an Arduino board. In this circuit the green and yellow alternate when blinking.







Fade (4C)


The circuit is built using two LEDs, resistors, four jumpers, a board and an Arduino board. In this circuit the green LED fades out while the yellow LED fades in.





Potmeter (6C)


The circuit is built using two LEDs, resistors, seven jumpers, a potmeter, a board and an Arduino board. A potmeter is a knob with variable resistance. that can be rotated. In this circuit the green LED fades out while the yellow LED fades in based on the values of the potmeter which were converted to serve as an input for the LED lights.





Light Matrix (7A)


The circuit is built using LEDs matrix resistors, five jumpers, a board and an Arduino board. A LED matrix is divided into rows and columns forming cells that can be controlled indivdually. In this circuit the LED Matrix shows a face that alternates between happy and a frowney by alternating the lighting of the cells.





Light (8A)


The circuit is built using two LEDs, three resistors with one of a different value, six jumpers, an LDR, a board and an Arduino board. An LDR is a variable resistor that measures the amount of light it receives. In this circuit LEDs LED fade based on the values read by the LDR which were converted to serve as an input for the LED lights.







Sound (9C)


The circuit is made with a 3 jumpers, one resistor, a Piezo meter, a board and an Arduino board. A Piezo speaker produces sound based on the input values. In this circuit a siren was created using 2 for loops.






Distance (11C)


The circuit is made with a LED, 5 jumpers, one resistor, an Ultrasonic Proximity Sensor, a board and an Arduino board. In this circuit the LED lights up when an object is detected within the specified range.






Processing1 (12D)


The circuit is built using two LEDs, resistors, seven jumpers, a potmeter, a board and an Arduino board. A connection is made between arduino and processing, through which the input of the potmeter on the Arduino is transfered to Processing chaging the size of the circle.






Processing2 (13B)


The circuit is a simple LED circuit that reacts to input form processing. When the 'P' on the keyboard is pressed, the LED on the circuit lights up.






Buttons (14B)


The circuit is built using six jumpers, an LED, two resistors, a button, a board and an Arduino board. When pressing on the light turns off.






Servemotor (15B)


The circuit is built using five jumpers, a servometer, a board and an Arduino board. A servometer is a motor with a mechanical arm and can be used to control vehicals. In this cicuit, the arm moves slowly clockwise and jolts back to its starting position.






Capacitive Touch (16D)


The circuit is built using six jumpers, two resistors, aluminium foil, a board and an Arduino board. When pressing on the aluminium foil the values on the serial screen change as the aluminium foilacts as a sensors providing input.






Reflection


It is interesting to explore the topic of Arduino and to gain basic practical experience. By learning a bit about it, it might be interesting to see how arduino can be used to automate things at home or to to create basic security devices, for instance by using the proximity sensor. Arduino can also serve as a tool for children to enagage and learn more about technology as it allows them to learn and and directly observe the results using a pyhsical device. In general, arduino provides a simple and easy way for those who are less technical to produce technical creations.
